# Trading Journal Pro

A comprehensive web-based trading journal application to track your trades, analyze performance, and maintain trading notes.

## Features

### 📊 Dashboard
- **Key Metrics**: Win rate, average R:R ratio, best/worst trades
- **Interactive Charts**: Overall P&L chart and weekly performance summary
- **Daily Performance Cards**: This week's performance by day
- **Monthly Calendar**: Visual calendar showing profitable/losing days
- **Date Filters**: Filter dashboard data by date range

### 📝 Trade Management
- **Add Trades**: Comprehensive trade entry form with:
  - Basic trade details (symbol, quantity, direction, prices)
  - Strategy selection
  - Tags for categorization
  - Psychology tracking (confidence, satisfaction, emotional state)
  - Mistake tracking and lessons learned
- **View All Trades**: Sortable table with filtering options
- **Edit/Delete Trades**: Modify existing trades or remove them

### 📋 Notes System
- **Multiple Notes**: Create, edit, and delete individual notes
- **Organized Display**: Notes shown with titles, dates, and content
- **Rich Content**: Support for multi-line text and formatting

### ⚙️ Settings
- **Strategy Management**: Add/remove trading strategies
- **Tag Management**: Customize trade tags
- **Risk Management**: Set default risk per trade
- **Data Management**: Export/import data, clear all data

## Getting Started

1. **Open the Application**: Open `index.html` in your web browser
2. **Add Demo Data**: Click "Add Demo Data" on the Notes page to see sample trades and notes
3. **Start Using**: Begin adding your own trades and notes

## Usage Guide

### Adding a Trade
1. Navigate to "Add Trade" section
2. Fill in the trade details (symbol, quantity, direction, prices)
3. Select strategy and add relevant tags
4. Complete psychology tracking section
5. Add lessons learned
6. Click "Add Trade"

### Creating Notes
1. Go to "Notes" section
2. Click "Add Note"
3. Enter title and content
4. Click "Save Note"

### Viewing Performance
1. Dashboard shows overall performance metrics
2. Use date filters to analyze specific periods
3. Charts provide visual performance insights
4. Calendar shows daily performance at a glance

### Managing Data
- **Export**: Use "Export JSON" to backup your data
- **Import**: Use "Import JSON" to restore from backup
- **Clear**: Use "Clear All Data" in settings to start fresh

## Data Storage

All data is stored locally in your browser's localStorage. This means:
- Data persists between browser sessions
- No internet connection required
- Data is private and secure
- Export/import for backup and transfer

## Browser Compatibility

Works best in modern browsers:
- Chrome (recommended)
- Firefox
- Safari
- Edge

## Features Overview

### Charts
- **Overall P&L Chart**: Shows cumulative profit/loss over time
- **Weekly Chart**: Displays daily P&L for the last 7 days
- **Theme Support**: Charts adapt to light/dark theme

### Psychology Tracking
- Entry confidence (1-10 scale)
- Satisfaction level (1-10 scale)
- Emotional state tracking
- Common mistake identification
- Lessons learned documentation

### Filtering & Sorting
- Filter trades by symbol, strategy, emotional state, tags, and date range
- Sort trades by any column
- Date range filtering for dashboard

### Mobile Responsive
- Works on desktop, tablet, and mobile devices
- Mobile navigation menu
- Responsive design adapts to screen size

## Tips for Best Results

1. **Be Consistent**: Add trades immediately after closing them
2. **Track Psychology**: Honest emotional state tracking helps identify patterns
3. **Use Tags**: Consistent tagging helps with analysis
4. **Regular Reviews**: Use the dashboard to review performance regularly
5. **Backup Data**: Export your data periodically
